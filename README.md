# find_wally

Find Wally from given image file using Tensorflow.

## Usage

```
$ find_wally -model /path/to/tensorflow-object-detection-model.pb wally-evaluate.jpg
```

## Requirements

Tensorflow 1.8

## Installation

```
$ go get github.com/mattn/find_wally
```
